ApexPlanet Web Development Internship
 
This project is submitted as Task 3 of the ApexPlanet Web Development Internship.
 
## 📌 Objective
 
The objective of Task 3 was to:
- Apply responsive design using CSS media queries
- Build an interactive JavaScript project (Quiz)
- Fetch and display real-time data from a public API
 
---
 
## 🚀 Project Overview
 
**DevToolkit** is a single-page web application that combines two developer utilities:
 
1. **GitHub Profile Finder** — Search any GitHub username and view their profile, stats, and top repositories using the GitHub REST API
2. **Web Dev Quiz** — An interactive 10-question quiz on HTML, CSS, and JavaScript with a countdown timer and score tracking
 
---
 
## 🛠 Technologies Used
 
- HTML5
- CSS3 (Flexbox, CSS Grid, CSS Variables, Media Queries)
- JavaScript (Vanilla JS — Fetch API, DOM Manipulation, Event Handling)

## 📂 Features Implemented
 
### GitHub Profile Finder
- Search GitHub users by username
- Displays avatar, name, bio, location, website, and join date
- Shows public repos, followers, following, and gists count
- Lists top 6 repositories with language, stars, and forks
- Loading spinner during API fetch
- Error handling for invalid usernames and rate limits
- Direct link to GitHub profile
 
### Web Dev Quiz
- 10 questions covering HTML, CSS, and JavaScript
- 20-second countdown timer per question (turns red at 5s)
- Progress bar showing quiz completion
- Instant correct/wrong answer feedback
- Score tracking and result screen with performance message
- Restart quiz option
 
### General
- Fully responsive layout for mobile, tablet, and desktop
- CSS media queries for breakpoints at 900px, 600px, and 400px
- Hamburger navigation menu for mobile
- Smooth scroll navigation with active link highlighting
- Dark theme using CSS variables
- Smooth animations and hover effects
