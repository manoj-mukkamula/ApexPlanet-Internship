# Task 2 – TaskFlow | Task Manager & Contact Form
ApexPlanet Web Development Internship

This project is submitted as Task 2 of the ApexPlanet Web Development Internship.

## 📌 Objective

The objective of Task 2 was to:
- Create a contact form using HTML and CSS with various input types
- Add JavaScript form validation for required fields and email format
- Build a responsive layout using Flexbox and CSS Grid with media queries
- Develop a dynamic to-do list using JavaScript DOM manipulation

---

## 🚀 Project Overview

**TaskFlow** is a lightweight task manager to organize your work with priority levels, category labels, and due dates — featuring a live stats dashboard and a fully validated contact form, built with pure HTML, CSS & JavaScript.

- Clean dark navy navbar with smooth active link highlighting
- Hero section with feature highlights
- Task manager with priority, category, due date, search, and sort
- Live stats sidebar with progress bar
- Fully validated contact form with inline errors and success feedback
- Fully responsive across desktop, tablet, and mobile

---

- CSS3 (Flexbox, CSS Grid, Media Queries, CSS Variables)

## 📂 Features Implemented

- Sticky navbar with Flexbox layout, active scroll spy, and mobile hamburger menu
- Hero section with CTA buttons and feature highlight cards
- Add tasks with title, description, priority (High/Medium/Low), category (Work/Personal/Other), and due date
- Inline JavaScript form validation — no `alert()`, errors appear per field
- Real-time task search and sort (Newest, Oldest, Priority, A→Z)
- Live stats panel showing Total, Completed, and Pending tasks with animated progress bar
- Task completion toggle with strikethrough and checkbox
- Task deletion with slide-out animation
- Toast notifications for every user action
- Overdue task detection with color highlight
- Contact form with name, email, subject, message fields
- Email format validation on blur + submit
- Character counter on message textarea
- Loading spinner on submit + success banner auto-hides after 6 seconds
- CSS Grid layout for main content area
- Responsive design with breakpoints at 1024px, 768px, and 480px
