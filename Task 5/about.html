About ShopEase

A capstone project demonstrating full e-commerce frontend development using HTML, CSS, and JavaScript : built as the final task of a 45-day web development internship.

What is ShopEase?

ShopEase is a fully functional e-commerce web application built from scratch using only HTML, CSS, and Vanilla JavaScript : no frameworks, no backend. It features a product listing page with live search, category filters, price range slider, sorting options, a product details modal, a shopping cart with quantity management, localStorage persistence, and a fully responsive design across mobile, tablet, and desktop.

Key Features

🛍️

24 Products

Across 5 categories : Electronics, Clothing, Footwear, Accessories, and Home & Living.

🔍

Smart Filters

Live search with debounce, category filter, price range slider, and 5 sort options.

🪟

Product Modal

Click any product to view full details : image, description, price, rating, and add to cart.

🛒

Shopping Cart

Add, remove, increase/decrease qty. Order summary with discount and free delivery logic.

💾

localStorage

Cart persists across page refreshes and navigation : no data loss.

📱

Responsive Design

Fully responsive at 480px, 768px, and 1024px breakpoints. Tested on Chrome, Firefox, and Safari.

Tech Stack
